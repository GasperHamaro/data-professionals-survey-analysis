# Data Professionals Survey Analysis (Power BI)

This project analyzes salary, job roles, and work satisfaction among data professionals using Power BI.

## Objective
To explore trends in salaries, job titles, and work satisfaction levels in the data industry.

## Tools Used
- Power BI
- Power Query (Data Cleaning in Power BI)
- Data Visualization

## Key Insights
- Data Scientists reported the highest average salaries.
- Data Analysts represented the largest share of respondents.
- Professionals with strong technical skills reported higher job satisfaction.
